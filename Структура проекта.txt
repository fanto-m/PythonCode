	|-repositories
	    |-__init__.py
	    |-base_repository.py Абстрактный базовый класс для всех репозиториев.
	                            Предоставляет общую функциональность для работы с SQLite:
	    |-categories_repository.py Предоставляет методы для:
                                                     - Создания и загрузки категорий
                                                     - Обновления и удаления категорий
                                                     - Генерации артикулов (SKU) для товаров
	    |-documents_repository.py  Предоставляет методы для:
                                                    - Добавления документов к товарам
                                                    - Загрузки документов товара
                                                    - Удаления и переименования документов
                                                    - Миграции старых документов
	    |-items_repository.py Предоставляет методы для:
                                                    - Создания и загрузки товаров
                                                    - Обновления и удаления товаров
                                                    - Поиска и фильтрации товаров
	    |-specifications_repository.py  Предоставляет методы для:
                                                    - Создания и загрузки спецификаций
                                                    - Обновления и удаления спецификаций
                                                    - Управления позициями в спецификациях
                                                    - Расчета стоимости спецификаций
	    |-suppliers_repository.py  Предоставляет методы для:
                                                    - Создания и загрузки поставщиков
                                                    - Обновления и удаления поставщиков
                                                    - Управления связями товар-поставщик
	    |-unit_of_work.py Предоставляет:
                                                    - Единую точку доступа ко всем репозиториям
                                                    - Автоматическую инициализацию всех таблиц БД
                                                    - Координацию работы между репозиториями
	|-utils
	    |-__init__.py
	    |-logger_config.py Настраивает логирование для приложения.

                                                    Args:
                                                        log_level: Уровень логирования (TRACE, DEBUG, INFO, WARNING, ERROR, CRITICAL).
                                                        log_dir: Директория для логов.
                                                        rotation: Когда создавать новый файл (размер или время).
                                                        retention: Как долго хранить старые логи.
                                                        compression: Сжимать ли старые логи.
